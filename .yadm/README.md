## Install X11
```
pacman -S xorg-xrdb xorg-xinit xorg-server
```


## Install yay
```
git clone https://aur.archlinux.org/yay.git
cd yay
makepkg -si
```

## Install AUR packages
```
yay -S ttf-google-sans openbox-patched dunst-git ruby-colorls ccat yadm-git
```
